Allows you to send Contact Form 7 messages to a group (admins, editors, authors, contributor, subscribers). 

== Description ==

Contact Form 7 Group Mail allows you to send messages to multiple users, according to their groups. With this plugin you can choose which groups (admins, editors, authors, contributors, subscribers) are going to receive the e-mails. The list of recipients is automatically updated, so you don't need to change the contact form every time a new user is added or removed. Important: it is an extension for Contact Form 7 plugin.

== Installation ==

1. Upload 'contact-form-7-group-mail' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Group Mail' page under 'Contact' menu and select groups to receive the e-mails.
